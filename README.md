# Cybersecurity Research Documentation

A repository containing cybersecurity research documents covering various attack techniques and security tools.

## 📂 Projects

### 🔍 NIKTO
Nikto is an open-source web server scanner that performs comprehensive tests against web servers for multiple security vulnerabilities. This document covers:
- Overview of the Nikto tool
- How Nikto scans work
- Findings and analysis from scans
- Potential vulnerabilities detected

### 💉 SQL Injection Attack
SQL Injection is one of the most common and dangerous web application vulnerabilities. This document covers:
- What SQL Injection attacks are
- Types of SQL Injection (Classic, Blind, Time-based)
- How attackers exploit SQL vulnerabilities
- Real-world attack examples and analysis
